# Example output

Three real review sheets generated by this pipeline, showing the full page structure —
One Thing hero, table of contents, step cards with key points / explanation / verbatim
transcript, and (where a source URL was set) click-to-watch links back to the video.

The extracted video frames themselves have been replaced with a placeholder image in
these copies, since redistributing frame grabs from someone else's instructional video
publicly is a different rights situation than using them for personal study. Run the
pipeline yourself on a video you have the rights to for the full illustrated version.
